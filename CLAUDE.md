# 项目踩坑记录

### 屏幕连接配置

| STM32H750 | ILI9341 | 说明      |
| --------- | ------- | --------- |
| PB15/MOSI | SDI     | 数据输入  |
| PB13/SCK  | SCL     | 时钟      |
| PB12/CS   | CS      | 片选      |
| PB14/MISO | SDO     | 数据输出  |
| PB1/RS    | D/C     | 数据/命令 |
| PB0/BLK   | BLK     | 背光控制  |

**注意**: 复位引脚(RST)直接接 VCC，使用软件复位初始化
