# 像素风格 UI 设计文档

**日期**: 2026-03-01
**风格**: 极简线框 + 扩展灰度 + Game Boy 风格
**主要用途**: 温湿度传感器数据展示

---

## 概述

重新设计嵌入式 UI，采用复古像素风格，专注于传感器数据的简洁展示。

### 设计目标

- 8 级灰度提供丰富层次
- 极简线框风格
- 中等像素字体 (10x14)
- Game Boy 风格方块图标
- DMA 优化保持高性能

## 颜色方案

### 8 级灰度定义

| 级别 | RGB 值 | 用途 |
|------|--------|------|
| g0 | (0, 0, 0) | 线条、文字 |
| g1 | (40, 40, 40) | 按下按钮填充 |
| g2 | (80, 80, 80) | 进度条填充 |
| g3 | (120, 120, 120) | 边框阴影 |
| g4 | (160, 160, 160) | 普通按钮填充 |
| g5 | (200, 200, 200) | 禁用状态、扫描线 |
| g6 | (230, 230, 230) | 辅助背景 |
| g7 | (255, 255, 255) | 主背景 |

```rust
pub struct GrayTheme {
    pub g0: Rgb565,  // 线条、文字
    pub g1: Rgb565,  // 按下按钮
    pub g2: Rgb565,  // 进度条填充
    pub g3: Rgb565,  // 边框阴影
    pub g4: Rgb565,  // 普通按钮
    pub g5: Rgb565,  // 禁用状态
    pub g6: Rgb565,  // 辅助背景
    pub g7: Rgb565,  // 主背景
}
```

## 像素图标

### 8x8 像素图标定义

```rust
#[derive(Clone, Copy)]
pub enum PixelIcon {
    Thermo,   // 温度计 🌡
    Humid,    // 湿度计 💧
}

impl PixelIcon {
    pub fn data(&self) -> &[u8; 8] {
        match self {
            Self::Thermo => &[
                0b00111100,
                0b01000010,
                0b01000010,
                0b00111100,
                0b00111100,
                0b00100100,
                0b00100100,
                0b00100100,
            ],
            Self::Humid => &[
                0b00100100,
                0b00100100,
                0b00100100,
                0b01111110,
                0b11111111,
                0b11111111,
                0b11111111,
                0b01101110,
            ],
        }
    }
}
```

## 扫描线效果

可选的 CRT 风格扫描线，隔行变暗：

```rust
for y in (1..height).step_by(2) {
    display.fill_solid(
        &Rectangle::new(Point::new(0, y), Size::new(width, 1)),
        theme.g5,  // 稍微变暗
    )?;
}
```

## DMA 优化

保持现有的 DMA 优化策略：
- 使用 `fill_solid` 进行大面积填充
- 脏矩形跟踪，只重绘变化区域
- 双缓冲区批量传输
